# 数据及其他材料说明

作品编号：TJJM202600000C

本材料用于配合论文《基于多源数据与空间计量的新质生产力虹吸与赋能效应测度》进行复现检查。

## 一、数据来源

1. 广东省统计年鉴：用于整理地区生产总值、常住人口、社会消费品零售总额、财政支出、金融机构存贷款、实际利用外资、研发投入、信息软件业主体等指标。
2. 各地级市统计年鉴和统计公报：用于补充个别城市—年份统计口径缺失项。
3. 上市公司年报文本：用于构造年报文本 AI 指数，并与官方统计代理指标进行交叉验证。
4. DataV.GeoAtlas 广东省地市边界数据：用于 LISA 空间聚类地图和空间可视化。
5. 城市中心经纬度公开资料：用于构建地理反距离矩阵、4 近邻矩阵和地理—经济嵌套矩阵。

## 二、压缩包内容

压缩包内保留处理后的核心数据、空间矩阵、建模脚本和表格生成脚本。原始广东统计年鉴 ZIP 文件体量较大，完整原始数据可按赛事系统要求另行通过云盘链接提交。

主要目录说明：

- `data/processed/`：清洗后的城市面板、空间权重、Moran/LISA 和 SDM 输出。
- `src/`：Python 数据清洗、指标测算、空间检验、制图和结果表生成脚本。
- `stata/`：Stata 面板 SDM 复核数据、空间矩阵和 `xsmle` 脚本。
- `paper_tables/`：论文正文和附录引用的 LaTeX 表格文件。
- `references.bib`：参考文献 BibTeX 文件。

## 三、复现建议

1. 运行 `src/pipeline/build_completed_guangdong_panel.py` 生成 21 市 2018—2023 年平衡面板。
2. 运行 `src/pipeline/run_panel_spatial_baseline.py` 生成空间权重矩阵、Moran's I 和 LISA 结果。
3. 运行 `src/pipeline/run_python_panel_sdm.py` 或 `stata/run_panel_sdm.do` 复核 Panel SDM 核心结果。
4. 运行 `src/pipeline/refresh_publication_figures.py` 更新论文图件。
